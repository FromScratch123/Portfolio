    Corange

    http://corange.work

    https://github.com/FromScratch123/Corange

    概要

    自作のWEBサービスです。 様々なジャンルのクリエイター同士が繋がり、一つのプラットフォーム上で共に仕事を進めていくことができるサービスを想定しています。 MVCモデルを採用し、開発を行っています。ユーザー管理機能、CRUD機能、Ajax技術を取り入れています。

    使用言語

    • HTML CSS
    • JavaScript (jQuery)
    • PHP
